# Python Mini Library

A Streamlit app that presents a searchable library of custom Python functions organized by category. Users can browse function details, view source code, and run functions with UI input fields.

## Features

- Search by function name or keyword
- Filter by category
- Display description, parameters, return value, and source code
- Execute functions with live UI inputs
- Gracefully report invalid input and runtime errors

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
